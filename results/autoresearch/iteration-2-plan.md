# 迭代 2 优化方案：结论可接受性维度

## 问题诊断

### 当前状态（v2.19 基线）
- 维度标准差：11.5 分
- 模型评分：
  - GLM-5.1: 92 分（共同体对话度 4/4，制度可接受度 4/4）
  - Kimi-K2.6: 72 分（共同体对话度 3/4，制度可接受度 3/4）
  - Qwen3.6-Plus: 92 分（共同体对话度 4/4，制度可接受度 4/4）

### Kimi-K2.6 评分偏低原因

**共同体对话度扣分（3/4）：**
- 标准3："结论在中国法学语境中说得通"
- Kimi 的判断："未充分讨论本土法学传统局限"
- 实际情况：论文已经讨论了中国民法典编纂语境，但 Kimi 要求更深入的"局限性讨论"

**制度可接受度扣分（3/4）：**
- 标准相对宽松，但 Kimi 仍然扣分
- 可能原因：对"制度转化方向"的要求过于具体

## 优化策略

### 方案 A：放宽"中国法学语境"标准（推荐）

**修改位置**：`conclusion_consensus` 维度的 prompt_template

**当前表述**（第 1186 行）：
```
□ 标准3：结论在中国法学语境中说得通
   满足条件：结论考虑了中国制度语境和法学传统，而非直接移植外国理论
   ✅ 成立示例：结论讨论了该观点在中国法治传统中的定位
   ❌ 不成立示例：直接移植外国制度/理论，不考虑本土适配
```

**优化后表述**：
```
□ 标准3：结论在中国法学语境中说得通
   满足条件：结论考虑了中国制度语境，而非直接移植外国理论
   【重要】只要结论提及中国制度背景（如民法典编纂、现行法条）即可，不要求深入讨论"本土法学传统局限"
   ✅ 成立示例：结论讨论了该观点在中国法治传统中的定位
   ✅ 成立示例：结论结合中国民法典编纂语境展开（即使未讨论局限性）
   ❌ 不成立示例：直接移植外国制度/理论，完全不考虑中国语境
```

**预期效果**：
- Kimi 共同体对话度：3/4 → 4/4
- Kimi 评分：72 → 85-92
- 结论可接受性 std：11.5 → <8

### 方案 B：增加"部分满足"的容忍度

**修改位置**：scoring_criteria 的 good 档标准

**当前标准**：
```
good:
  range: [60, 79]
  indicators:
    - "共同体对话度≥2且制度可接受度≥2"
```

**优化后标准**：
```
good:
  range: [60, 79]
  indicators:
    - "共同体对话度≥2且制度可接受度≥2"
    - "共同体对话度=3且制度可接受度=3 → good 高段（75-79）"
```

**预期效果**：
- Kimi 评分：72 → 75-79（仍在 good 档，但更接近 excellent）
- 结论可接受性 std：11.5 → 9-10

## 推荐方案

**选择方案 A**，原因：
1. 直接解决根本问题（Kimi 对"中国法学语境"的理解过严）
2. 不改变评分档位，只调整标准的宽严度
3. 更符合实际论文的写作习惯（不是所有论文都会深入讨论"局限性"）

## 实施步骤

1. 复制 v2.21 → v2.22（假设 v2.21 验证通过）
2. 修改 `conclusion_consensus` 维度的标准3表述
3. 增加【重要】提示，明确不要求"深入讨论局限性"
4. 增加正面示例："结合中国民法典编纂语境展开"
5. Git 提交并运行验证
6. 分析结果，判断是否改进

## 风险评估

**风险**：放宽标准可能导致对"直接移植外国理论"的论文过于宽容

**缓解措施**：
- 保留反面示例："完全不考虑中国语境"
- 只放宽"深入讨论局限性"的要求，不放宽"考虑中国语境"的基本要求
- 通过验证测试确认不会误判

## 备选方案

如果方案 A 效果不佳，可以尝试：
1. 同时优化"制度可接受度"的标准
2. 调整锚定表，降低 good 档的下限
3. 增加"理论论文"和"制度论文"的差异化评分标准
