### 现状洞察度 (literature_insight)

**std = 16.3**, confidence = critical
模型分数：gpt-5.4=62, qwen3.6-plus=92, glm-5.1=88

**根因诊断**: 本维度分歧的核心不在总分表面，而在两个底层判断口径不一致：一是“法学流派”究竟按明确命名的对立立场计数，还是按围绕同一争点的不同解释路径宽泛计数；二是“三要素充分满足”是否必须出现可逐条对应的明示文本证据，还是可由分散表述综合推断。当前 prompt 一方面给了较宽的流派定义（含‘代表性学术立场’），另一方面又给了少量强制认定组合，导致保守模型只认明确流派名词、宽松模型则把多个作者路径都算作独立流派。另一个问题是锚定表与 scoring_criteria 的分档区间并不完全一致，且 excellent/good 档内部缺少细化落点规则，放大了模型在 85-92 与 62-74 之间的自由裁量。

**风险评估**: 这些改动主要收紧“流派计数”和“充分满足”的证据门槛，预计会降低 literature_insight 的虚高分并压缩 90+ 的比例，评分分布可能从右偏改为更集中于 65-85 区间。对其他维度影响较小，因为修改集中在本维度专属 prompt 与规则，但“法律道德主义是否可在未明示对立面时自动补足分离论”这一点涉及法学理论口径，建议由法学专家最终确认。

### 逻辑严密性 (logical_coherence)

**std = 5.1**, confidence = medium
模型分数：gpt-5.4=72, qwen3.6-plus=62, glm-5.1=65

**根因诊断**: 分歧主要来自三个锚点解释不一致：一是“主线连贯”与“可严密推出”之间的门槛未分开，导致有模型按结构顺畅给到 good，另一些模型按规范证成缺口判为 marginal；二是现有 prompt 只要求做“链条复述测试/结论推出测试”，但没有明确“规范桥梁缺失”“概念重定义替代论证”是否直接构成关键跳步；三是 ceiling rule 中“至少2处明显跳步”的计数标准过粗，模型对一处核心跳步是否足以压到 50 以下理解不一，尤其在法学论文中，核心规范结论的单一关键断裂往往比多处局部跳步更严重。
