# 关于学术评价锚点、人机分工与古德哈特困境的几点看法

我倾向于把这三个问题放在同一条主线上看：`SocialEval` 不是在追求“纯客观评价”，而是在追求“可审计、可复核、可约束的准客观评价”。这比“AI 能否替代学者判断”更接近项目真实目标。

## 问题一：内容评价的“锚”在哪里，是否存在可操作的客观标准？

有，但不是“绝对客观标准”，而是“分层客观性”。

我建议把“锚”分成三层：

1. `文本内锚`

   指论文本身能被定位和引用的事实，例如研究问题是否明确、论证链是否闭合、是否回应核心反对意见、文献引用是否支撑关键主张。这一层最适合 AI，因为可以要求它必须给出 `evidence_quotes`，把评分绑定到原文片段。

2. `共同体锚`

   指某学科当前可辨识的评价共识，例如法学论文通常会看问题意识、文献掌握、理论建构、论证严密性、与主流争论的对话、议题延展性。需求中的六维框架，本质上就是把“隐性的同行评议经验”外显成结构化标准。

3. `程序锚`

   指不是问“这分数绝对对不对”，而是问“这个分数是否按同一程序得出，是否可追溯、可比较、可复核”。多模型并发、均值/标准差、低置信度进专家队列，都是程序锚。

所以，学术评价的可操作客观性，不在于消灭判断，而在于把判断约束进一套公开程序里。真正不可完全客观的，是“原创性有多高”“理论贡献有多大”这类高阶判断；但它们仍然可以被“证据化、结构化、复核化”。

我的判断是：

- `可操作的客观标准存在`
- `但只能做到有限客观，而非绝对客观`
- `越靠近文本证据，客观性越强；越靠近学术价值判断，主观性越高`

## 问题二：AI 辅助评价中，人和机器的分工边界在哪里？

最稳妥的边界不是“AI 做初审，人做终审”这么粗，而是按任务类型切。

机器更适合：

- 结构化拆解
- 跨维度一致执行同一 rubric
- 从全文中抽取证据片段
- 生成维度级解释
- 多模型对比和分歧检测
- 发现明显缺口，比如文献综述薄弱、论证跳步、概念定义不清

人更适合：

- 判断“这个问题值不值得研究”
- 判断“创新到底是真创新，还是换说法”
- 判断“是否触及领域前沿争议”
- 识别文本之外的学术语境、范式差异、隐含立场
- 对低置信度、边界案例、范式创新型论文做最终裁断

换句话说：

- `AI 负责把评价过程显性化、标准化、证据化`
- `人负责处理价值判断、范式判断、例外判断和责任承担`

我不建议把人机边界设成“AI 给分，人只签字”。这会把专家退化成背书者。更合理的是：

- 高置信度论文：AI 提供稳定初评，人工抽检
- 低置信度论文：人工重点介入
- 高分且高分歧论文：必须人工介入
- 明显偏离常规范式但有潜在创新的论文：必须人工介入

也就是说，人不该平均介入所有样本，而应该集中介入“AI 最不该独断”的样本。

## 问题三：AI 评价是否会催生新一轮“古德哈特困境”？

会，而且几乎是必然的。只要评价指标变成可见、可优化的目标，就会出现“为了拿高分而写”的逆向适配。

在这个场景里，风险主要有四类：

1. `维度投机`

   作者开始迎合六维框架的话术，把“问题创新性”“理论建构力”写成模板化姿态，而不是真的有贡献。

2. `证据投机`

   因为 AI 依赖可引用片段，作者会更刻意地制造“像证据的句子”，让文本显得更容易被模型抓取和高估。

3. `模型投机`

   如果大家逐渐知道系统偏好什么表达风格、篇章结构、引用密度，就会出现针对 AI 评审器的写作优化，而不是针对学术问题本身优化。

4. `制度投机`

   编辑或机构一旦把 AI 分数当成强 KPI，系统原本的“辅助评价”会滑向“分数治理”，这时古德哈特效应最严重。

但这不意味着不能做，而是意味着系统设计必须把“反古德哈特”当成一等公民，而不是后补功能。

我建议项目层面至少坚持四条：

- `分数不能裸奔`
  任何分数都必须绑定证据片段、分析说明、模型分歧数据，避免单一数值成为唯一目标。
- `高分不等于直接通过`
  尤其对高分但分歧大的论文，不能因为均值高就自动放行。
- `定期轮换 rubric 与 prompt`
  如果评价维度描述、提示词、阈值长期固定，系统更容易被反向学习和适配。
- `把“异常一致”也当风险信号`
  不仅要盯低置信度，也要盯“异常工整、处处踩中 rubric”的文本，这类样本可能是在迎合系统。

我的总体判断是：

- `AI 评价不会消灭古德哈特困境`
- `它会把旧的主观性问题，转换成新的指标操纵问题`
- `但如果系统把分数、证据、分歧、人工复核绑定在一起，风险是可控的`

## 结论

`SocialEval` 最不该承诺的是“客观评分”，最该承诺的是“受约束的评价程序”。因为学术评价的终局不是找到一个永远正确的分数，而是建立一套让分歧可见、理由可查、偏差可纠正的机制。
